## Overview

She Safety is an innovative web application designed to leverage the power of the Google Gemini API for personal security. It provides an intuitive, responsive, and safe digital environment. The application uses modern web technologies including Vite, TypeScript, and React, combined with Firebase for robust backend services, ensuring a seamless and reliable experience.

## 💻 Run Locally

To get a local copy up and running, follow these simple steps.

**Prerequisites:** Node.js

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Environment Variables:**
   Set the `GEMINI_API_KEY` in your `.env.local` file (see `.env.example`) to your Gemini API key:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```
3. **Run the app:**
   ```bash
   npm run dev
   ```
