# 🛡️ She Safety

Welcome to **She Safety**, an intelligent web application designed to empower and protect. Built with the power of modern AI using Google's Gemini API, this project aims to provide a responsive and safe digital environment.

🚀 **Live Demo:** [https://she-safety.vercel.app/](https://she-safety.vercel.app/)

✨ **View in AI Studio:** [https://ai.studio/apps/8aac5e9f-3b5f-4f53-a168-84469b66da5b](https://ai.studio/apps/8aac5e9f-3b5f-4f53-a168-84469b66da5b)

## 🛠️ Built With

- **Frontend:** Vite
- **AI Integration:** Gemini API
- **Deployment:** Vercel

## 💻 Run Locally

To get a local copy up and running, follow these simple steps:

**Prerequisites:** Node.js

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Environment Variables:**
   Set the `GEMINI_API_KEY` in your `.env.local` file (see `.env.example`) to your Gemini API key:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```
3. **Run the app:**
   ```bash
   npm run dev
   ```
